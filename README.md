### HOW TO RUN
### Linux (I coded from ubuntu)
'sudo apt install g++ -y'
'cd ~'
'git clone https://github.com/Frixxions/cpp-smcalculator'
'cd smcalculator'
'g++ main.cpp -o smcalculator'
'cd cpp-smcalculator'
'./smcalculator'

### Windows 7 - 10 - 11
